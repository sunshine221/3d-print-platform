# 3D 打印产品展示与管理平台 — 需求文档

> 版本: v1.1 | 日期: 2026-05-11 | 状态: 待评审

---

## 1. 项目概述

### 1.1 项目背景

面向 3D 打印服务商/工作室，打造一个集**对外产品展示**与**内部业务管理**于一体的 Web 平台。
核心业务分为两条线：**SKU 下单**（标准产品选规格直接下单）和**代打服务**（客户上传模型文件，管理员报价后面议确认）。
客户可在线浏览产品目录、查看 3D 模型预览、选择 SKU 下单或提交代打需求；管理员通过后台管理产品、SKU、订单、代打询价、客户与内容。

### 1.2 项目目标

- 搭建品牌化的公开展示窗口，提升产品曝光与客户转化
- 提供 3D 模型在线预览，增强客户决策信心
- 统一管理产品、订单、客户数据，提升运营效率

### 1.3 用户角色

| 角色 | 说明 |
|------|------|
| 游客 | 浏览产品目录，无需登录 |
| 注册客户 | 可下单购买、提交代打需求、查看历史订单和询价记录 |
| 运营/编辑 | 管理产品、内容、查看询价 |
| 管理员 | 全权限：用户管理、系统配置、数据统计 |

## 2. 对外产品目录（前台）

### 2.1 首页

- **Banner 轮播**：展示主打产品、促销活动、品牌价值（多图 + CTA 按钮）
- **热门分类快捷入口**：图标 + 名称，如「手办模型」「工业零件」「建筑模型」「艺术雕塑」
- **精选产品推荐**：卡片流，显示缩略图、名称、材质、起售价
- **品牌价值条**：三项核心卖点（如「高精度打印」「多种材质可选」「快速交付」）
- **页脚**：联系方式、社交媒体链接、版权信息

### 2.2 产品浏览页

- **左侧筛选栏**：
  - 分类（树形，支持多级）
  - 材质（树脂、PLA、尼龙、金属等）
  - 工艺（SLA、FDM、SLS、MJF 等）
  - 价格区间
  - 精度/公差范围
  - 颜色
- **顶部搜索 + 排序**（默认、最新、价格升/降、热度）
- **产品卡片网格**：
  - 缩略图（hover 预览第二张或播放 3D 预览动图）
  - 产品名称、简要描述
  - 材质标签、起售价、「查看详情」链接
- **分页**

### 2.3 产品详情页

- **左侧：3D 模型查看器**（Three.js 实现）
  - 旋转 / 缩放 / 平移
  - 切换材质/颜色预览
  - 自动旋转展示
  - 全屏模式
- **右侧：产品信息与下单**
  - 名称、描述（富文本）、规格参数表
  - **SKU 选择器**：以规格维度展示（如材质 + 颜色 + 尺寸组合），每个 SKU 显示独立价格；支持图文标签选择
  - 选中 SKU 后展示：单价、起订量、交货周期、库存/产能状态
  - 数量输入框 + 备注文本框
  - 价格实时计算展示（数量 × SKU 单价）
  - 打印成品实拍相册（灯箱放大）
  - **双入口按钮**：
    - 「立即下单」— 走 SKU 下单流程（必选 SKU，固定价格）
    - 「代打服务」— 走上传模型面议流程（不需要 SKU）
- **底部：相关产品推荐**

### 2.4 SKU 下单流程

> 适用于标准化产品：客户选定 SKU 后按固定价格下单。

1. 产品详情页选择 SKU（规格组合），输入数量、填写备注
2. 点击「立即下单」，系统校验 SKU + 数量 + 起订量
3. 未登录跳转登录/注册；已登录进入订单确认页
4. **订单确认页**：
   - 核对产品信息、SKU 规格、数量、单价、总价
   - 填写收货信息（联系人、电话、地址）
   - 如有优惠码可在此输入
   - 提交订单
5. 提交后生成订单，显示订单编号和预计交付时间
6. 客户可在「个人中心 > 我的订单」查看订单状态（待确认/生产中/已发货/已完成/已取消）

### 2.5 代打服务流程

> 适用于非标需求：客户上传模型文件，管理员审核后报价，双方沟通确认。

1. 产品详情页或独立入口点击「代打服务」
2. **代打需求表单**：
   - 上传模型文件（STL / STEP / OBJ / 3MF，支持多文件，单文件最大 100MB）
   - 期望材质、颜色、数量、尺寸（可填写范围）
   - 期望交付时间
   - 补充说明（用途、精度要求、后处理等）
   - 联系方式（自动填充已注册用户的默认信息）
3. 提交后生成代打询价单，显示编号
4. **价格面议流程**：
   - 管理员后台审核模型文件，评估可行性与报价
   - 管理员填写报价（单价/总价）、备注、预计交付时间后提交
   - 客户端可见报价，可接受、拒绝或继续沟通（留言）
   - 客户接受报价后转为正式订单，进入订单流程
5. 客户可在「个人中心 > 我的代打」查看进度（待审核/已报价/沟通中/已接受/已拒绝/已关闭）

### 2.6 用户认证

- 注册（邮箱 + 密码 + 验证码 / 手机号）
- 登录、忘记密码
- 个人中心：基本信息、我的订单、我的代打、修改密码

### 2.7 其他页面

- **关于我们**：公司/工作室介绍
- **材料介绍**：各材质特性、对比表
- **打印指南**：FAQ、文件准备说明
- **联系我们**：表单 + 地图 + 联系信息

## 3. 后台管理系统（后台）

### 3.1 仪表盘

- 统计卡片：今日访问量、本月订单数、代打询价数、成交率、在售产品数
- 近期订单列表（最新 10 条，带状态标签）
- 近期代打询价列表（最新 10 条，带状态标签）
- 访问趋势折线图（近 30 天）
- 订单/询价来源分布（按产品 / 按分类）

### 3.2 产品管理

- **产品列表**：表格 + 搜索 + 分类筛选 + 批量上下架
- **新增/编辑产品**：
  - 基本信息：名称、副标题、描述（富文本）
  - 分类（多选末级分类）
  - 规格参数：动态添加行（参数名 + 参数值）
  - **SKU 管理**（一对多，每个产品下管理多个 SKU）：
    - SKU 编码（自动生成 + 手动修改）
    - 规格组合（从规格参数中选择，如「树脂 + 白色 + 15cm」）
    - 单价设置（支持固定价格）
    - 起订量、库存/产能状态
    - SKU 图片（可覆盖产品默认图）
    - 状态：启用 / 停用
  - 图片上传（多图，支持拖拽排序，设置首图）
  - 3D 模型文件上传（glTF/GLB 格式，限制大小）
  - SEO 信息：标题、描述、关键词
  - 状态：草稿 / 已上架 / 已下架
  - **产品类型**：标准产品（走 SKU 下单）/ 代打服务（仅走代打流程）/ 双模式（两者都支持）
- **批量操作**：上下架、修改分类、删除

### 3.3 分类管理

- 树形分类（支持三级）
- 每个分类：名称、图标、描述、排序、显示/隐藏
- 拖拽排序

### 3.4 订单管理

> 管理「SKU 下单」产生的正式订单。

- **订单列表**：表格 + 状态筛选 + 时间范围搜索
  - 列：订单编号、客户、产品 + SKU 规格、数量、总价、状态、创建时间
- **订单详情**：
  - 客户信息、收货信息
  - 产品信息、SKU 规格明细、数量、单价、总价
  - 客户备注
  - **状态流转**：待确认 → 生产中 → 已发货（可填写物流信息）→ 已完成 / 已取消
  - 操作日志（状态变更时间线）

### 3.5 代打询价管理

> 管理「代打服务」提交的询价单。

- **询价列表**：表格 + 状态筛选 + 时间范围搜索
  - 列：编号、客户、关联产品（可选）、期望材质/尺寸、状态、创建时间
- **询价详情**：
  - 客户信息、需求描述
  - 模型文件下载与预览（3D 查看器，支持 STL/OBJ 预览）
  - **管理员报价**：填写单价/总价、备注、预计交付时间，提交后客户端可见
  - **沟通记录**：客户与管理员可互发消息（留言板形式）
  - 状态流转：待审核 → 已报价 → 沟通中 → 已接受（转正式订单）→ 已拒绝 / 已关闭
  - 客户接受报价后，自动生成正式订单，进入订单管理流程
  - 操作日志（状态变更时间线）

### 3.6 客户管理

- 注册客户列表：搜索、查看详情
- 客户详情：基本信息、注册时间、历史询价、成交统计
- 禁用/启用账号

### 3.7 内容管理

- **首页 Banner**：图片 + 标题 + 链接 + 排序 + 启用/停用
- **页面管理**：关于我们、材料介绍、打印指南等可编辑页面（富文本编辑器）
- **联系我们配置**：电话、邮箱、地址、地图坐标

### 3.8 素材库

- 统一管理上传的图片和 3D 模型文件
- 支持文件夹分类、搜索、预览、删除
- 产品编辑时可直接从素材库选择

### 3.9 系统设置

- **基础设置**：站点名称、Logo、favicon、SEO 默认值
- **邮件配置**：SMTP 设置、邮件模板（询价通知等）
- **用户与角色**：角色列表（权限矩阵）、管理员账号管理
- **操作日志**：记录关键操作（谁、何时、做了什么）

## 4. 非功能性需求

| 维度 | 要求 |
|------|------|
| 性能 | 首屏加载 < 2s，3D 模型加载 < 5s（4G 网络），接口响应 < 500ms (P95) |
| 安全 | HTTPS 强制、密码加密存储 (bcrypt)、XSS/CSRF/SQL 注入防护、文件上传校验、RBAC 权限控制 |
| 可用性 | 响应式布局（PC + 平板 + 手机）、主流浏览器兼容（Chrome/Firefox/Safari/Edge 近两年版本） |
| 可维护性 | 前后端分离、RESTful API、代码规范 + 类型检查 |
| 可扩展性 | 模块化设计，预留多语言、多站点、支付对接扩展点 |

## 5. 技术选型建议

| 层 | 技术 | 说明 |
|----|------|------|
| 前端（前台） | Next.js 14 (App Router) + TypeScript + Tailwind CSS | SEO 友好，首屏快 |
| 前端（后台） | React 18 + TypeScript + Ant Design / shadcn/ui | 组件丰富，开发效率高 |
| 3D 渲染 | Three.js + React Three Fiber | glTF/GLB 模型预览 |
| 后端 | Node.js (NestJS) 或 Go (Gin) | 结构化框架，适合企业级 |
| 数据库 | PostgreSQL | 关系型，支持 JSON 字段 |
| 缓存 | Redis | 会话、热点数据 |
| 文件存储 | MinIO / 阿里云 OSS / AWS S3 | 图片 + 3D 模型文件 |
| 搜索引擎 | Elasticsearch (可选) 或 PostgreSQL 全文搜索 | 产品搜索 |

## 6. 数据模型概览

核心实体关系：

```
Category (分类)
  ├── parent_id (自引用，树形)
  └── Product[] (多对多)

Product (产品)
  ├── Category[] (多对多)
  ├── product_type: 标准产品 / 代打服务 / 双模式
  ├── specParams[] (规格参数定义)
  ├── SKU[] (一对多: 规格组合+单价+库存状态)
  ├── ProductImage[] (一对多: 图片URL+排序)
  ├── Model3D (一对一: glTF文件URL)
  └── Inquiry[] / Order[]

SKU (库存量单位)
  ├── Product (多对一)
  ├── sku_code / spec_combo (规格组合JSON: {"材质":"树脂","颜色":"白","尺寸":"15cm"})
  ├── price (固定单价)
  ├── min_order_qty / stock_status / image / status
  └── OrderItem[]

Order (订单 — SKU 下单)
  ├── order_no / Customer
  ├── OrderItem[] (一对多: SKU + 数量 + 单价快照)
  ├── total_price / note / shipping_info
  ├── status: 待确认 → 生产中 → 已发货 → 已完成 / 已取消
  └── OrderLog[] (操作时间线)

PrintServiceInquiry (代打询价)
  ├── inquiry_no / Customer / Product (可选)
  ├── model_files[] (STL/STEP/OBJ/3MF)
  ├── desired_material / color / quantity / size_range / deadline
  ├── admin_quote (报价: 单价/总价/备注/预计交付)
  ├── messages[] (沟通记录)
  ├── status: 待审核 → 已报价 → 沟通中 → 已接受 → 转Order / 已拒绝 / 已关闭
  └── InquiryLog[] (操作时间线)

Customer (客户)
  ├── Order[]
  └── PrintServiceInquiry[]

Banner / Page / MediaLibrary / SystemConfig / AdminUser / Role / OperationLog
```
